Data Deletion Instructions

Last updated: June 17, 2026

Ibis does not store any user data on external servers. All data is stored locally on your device within Adobe Lightroom's plugin preferences. No account is created, no personal data is collected remotely, and no images are retained after processing.

How to Delete All Ibis Data

To completely remove all data associated with Ibis from your device:

  1. Open Adobe Lightroom Classic
  2. Go to File > Plug-in Manager
  3. Select Ibis Metadata in the left panel
  4. Click Remove
  5. Confirm the removal when prompted

This permanently deletes all locally stored preferences, including your API key and output limit settings.

What Gets Deleted

  • Your stored Anthropic API key
  • Output limit preferences (max keywords, title words, description sentences)
  • Any temporary image files (these are also automatically deleted after each processing session)

What Was Never Stored

  • Your original photographs — never uploaded or copied
  • Personal information (name, email, address) — never collected
  • Usage data or analytics — never tracked
  • Generated metadata history — not retained by Ibis

Facebook / Meta Data

If you connected Ibis Publisher through Facebook, you can also remove the app connection from your Facebook account:

  1. Go to Facebook Settings > Apps and Websites
  2. Find Ibis Publisher and click Remove
